# Multi-LLM Reasoning System

複数のLLMモデルを組み合わせて推論するシステムのハンズオン

## 概要

このハンズオンでは、agent-frameworkを使用して複数のLLMモデルを組み合わせ、
ユーザーが与えた問いに対して高度な推論を行うシステムを構築します。

## 学習内容

- agent-frameworkの基本的な使い方
- 複数のLLMモデルの連携方法
- 推論プロセスの設計と実装
- エージェント間の協調動作

## 前提条件

- Python 3.11以上
- agent-framework 1.0.0b251104以上
- 各種LLMサービスのAPIキー

## セットアップ

```bash
# 仮想環境の有効化
source .venv/bin/activate

# 必要なパッケージは既にインストール済み
```

## 実装予定の機能

1. マルチエージェントシステムの構築
2. LLMモデル間の協調推論
3. ユーザークエリの処理と応答生成
